How to create dlnasrv sis package:
--------------------------------------------------------------------------------
1. Run dlnasrv\install\make_sis.cmd
2. Sign dlnasrv.sis
3. Sign dlnasrv_cenrep_keys.sis

NOTE: Both dlnasrv.sis and dlnasrv_cenrep_keys.sis are required to be able to use
dlna services!
